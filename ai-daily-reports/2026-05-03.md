- **来源:** AI News Roundup / thinkpol.ca
- **摘要:** 中国开源模型 Kimi K2.6（Moonshot AI）在 AI 编程竞赛 Word Gem Puzzle 中以 22 分（7-1-0 战绩）夺冠。GPT-5.5 排名第三，Claude Opus 4.7 第五，Gemini Pro 3.1 第六。小米 MiMo V2-Pro 排名第二。这显示中国开源模型在编程能力上已达到甚至超越西方前沿模型的水平。Kimi 的策略是激进的贪心算法——持续滑动滑块解锁新词，在大规模网格上累积得分最高。

### 2. 马里兰州拟禁止 AI 驱动的超市动态定价
- **来源:** AI News Roundup / 纽约时报
- **摘要:** 马里兰州计划立法禁止超市使用 AI 驱动的"监控定价"（surveillance pricing），即根据消费者行为数据实时调整商品价格的算法。这是美国首个针对 AI 定价的州级禁令，反映了 AI 在商业场景中引发监管关注的趋势。
- **链接:** https://www.nytimes.com/2026/05/01/business/surveillance-pricing-groceries-maryland.html

### 3. AI 亲密设备与隐私危机：那些你不想分享的数据
- **来源:** AI News Roundup / fshot.org
- **摘要:** 配备生物反馈传感器的 AI 亲密设备正在普及（售价约 20 英镑），这些设备会实时学习和适应用户偏好。但问题在于：这些高度私密的生物数据会被发送到远程服务器，其存储位置、访问权限和安全性均不透明。文章警告，亲密生物数据可能成为数据经纪商的下一个目标。
- **链接:** https://fshot.org/techzone/the-algorithm-knows.php

### 4. Specsmaxxing：用 YAML 写规范，克服"AI 精神错乱"
- **来源:** AI News Roundup / acai.sh
- **摘要:** 开发者分享了用 YAML 编写 AI 编码规范的方法论，称"Peak Slop"（AI 生成垃圾代码的高峰）已经过去，我们正在进入"后 slop 时代"。核心观点：AI 编码的瓶颈不是想象力，而是上下文窗口；用结构化规范（而非自然语言）向 AI 传达需求，可以显著减少反复迭代。作者还开源了一套相关工具。
- **链接:** https://acai.sh/blog/specsmaxxing

### 5. VS Code 自动添加 'Co-Authored-by Copilot' 引发争议
- **来源:** AI News Roundup / GitHub
- **摘要:** VS Code 被发现无论是否实际使用 Copilot，都会在 git commit 中自动插入 "Co-Authored-by: GitHub Copilot" 标签，引发开发者社区不满。PR #310226 正在讨论此问题。
- **链接:** https://github.com/microsoft/vscode/pull/310226
