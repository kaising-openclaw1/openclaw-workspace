## 🔥 今日重点

### 1. OpenAI 推出 ChatGPT 个人理财功能，支持连接银行账户
**来源：TechCrunch AI**

OpenAI 面向美国 ChatGPT Pro 用户推出个人理财工具预览版，通过与 Plaid 合作，支持连接超过 12,000 家金融机构（Schwab、Fidelity、Chase、Robinhood 等）。用户可查看投资组合表现、支出分析、订阅管理和即将到期的付款。该功能建立在 4 月收购个人理财 AI 初创公司 Hiro 的团队基础之上。每月已有超 2 亿用户向 ChatGPT 提出财务问题。OpenAI 表示新的 GPT-5.5 模型在上下文推理方面更强，对金融类问题至关重要。

### 2. OpenAI 与 Apple 合作关系破裂，正在探索法律选项
**来源：Ars Technica AI**

据 Bloomberg 报道，OpenAI 对 Apple 在 iOS 中集成 ChatGPT 的方式感到"被烧"。OpenAI 原预期该合作"每年可带来数十亿美元订阅收入"，但 Apple 故意未充分推广该功能。集成设计存在严重问题：用户必须特别说出"ChatGPT"才能调用，且返回窗口极小。OpenAI 认为这损害了 ChatGPT 品牌形象，正在考虑法律途径。

### 3. Anthropic $15 亿版权和解案遭法官延期批准
**来源：Ars Technica AI**

美国联邦法官 Araceli Martinez-Olguin 拒绝批准美国史上最大版权和解案。部分作者反对律师团队要求超过 3.2 亿美元律师费，而每位作者仅预期获得约 3,000 美元赔偿。反对者指出律师费率相当于每小时 10,000–12,000 美元，极为不合理。和解案最终批准被推迟，等待进一步听证。

### 4. OpenAI 再次大改组，全力押注 AI Agent
**来源：The Verge AI**

OpenAI 周五宣布新一轮组织架构调整，合并产品团队，由 Greg Brockman 全面负责产品战略。核心举措是将 ChatGPT 和 Codex 合并为"统一的 Agent 体验"。此举是在 AGI 负责人 Fidji Simo 休假后的持续调整，OpenAI 正聚焦编码和企业等核心收入来源，砍掉"副业项目"，为今年可能的 IPO 做准备。

### 5. Anthropic 提升 Claude Code 用量限制，与 SpaceX 达成计算合作
**来源：Ars Technica AI**

Anthropic 在 Code with Claude 2026 大会上宣布：Pro 和 Max 计划用户的 Claude Code 用量翻倍，并公布与 SpaceX 的计算合作。Anthropic CEO Dario Amodei 表示公司"按年增长 10 倍的规模做了充分准备"，但 Claude Code 的实际增长仍然超出预期。Cat Wu（Claude Code 产品负责人）表示没有长期路线图，而是根据开发者反馈和模型能力提升来调整方向。

### 6. Claude 的企业之战不在模型，而在 Agent 控制平面
**来源：VentureBeat AI**

Anthropic 认为企业市场的下一个竞争焦点不是基础模型，而是 Agent 的控制层（control plane）——即如何编排、监控和管理多个 AI Agent 协同工作。

### 7. RecursiveMAS：多 Agent 推理加速 2.4 倍，Token 消耗降低 75%
**来源：VentureBeat AI**

新技术 RecursiveMAS 可显著提升多 Agent 系统的推理效率，速度提升 2.4 倍，Token 使用量减少 75%，对降低企业 AI 部署成本意义重大。

### 8. Runway：从影视工具到世界模型，要挑战 Google
**来源：TechCrunch AI**

AI 视频生成公司 Runway 透露其战略方向：从语言模型转向"世界模型"——直接从视频和观察数据中学习世界的运作规律，而非仅从文本中学习。联合创始人认为这是 AI 的下一个前沿。

### 9. Tesla 披露两起 Robotaxi 远程操作员相关事故

5. **计算成本仍是瓶颈**：Anthropic 被迫提升用量限制，多 Agent 优化方案受关注——AI 扩展仍受制于算力资源。
