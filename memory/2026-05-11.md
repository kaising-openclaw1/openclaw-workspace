#### 1. 差分截屏模块（`agent/diff_capture.py`）
- 将屏幕分割为 64×64 像素块
- 对比前后帧 MD5，仅传输变化区域
- 预计节省 70-90% 带宽
- 变化超过 40% 自动回退完整帧（更高效）

#### 2. 连接码系统（`agent/connection_card.py`）
- 9 位数字 ID + 6 位临时密码
- 类似 TeamViewer 的连接方式
- 10 分钟自动过期
- 支持手动刷新

#### 3. 会话录制模块（`agent/session_record.py`）
- 自动录制远程控制会话
- 保存为 `.reml` 格式（JSON 行）
- 包含截屏帧和输入事件时间线
- 支持回放功能
- 录制文件存储在 `~/.remoteeye/recordings/`

#### 4. 远程 Shell 功能
- Agent 端启动 bash 子进程
- 通过 WebSocket 双向通信
- Web 端实时显示输出和输入
- 适合运维场景

#### 5. 端到端加密（`agent/e2e_crypto.py`）
- AES-256-GCM 加密
- 支持 Diffie-Hellman 密钥交换
- PIN 码派生会话密钥

#### 6. 连接质量监控（`agent/quality_monitor.py`）
- 实时测量延迟（RTT）
- 帧率统计
- 带宽利用率
- 综合质量评分 0-100
- 质量等级：优秀/良好/一般/较差

#### 7. H.264 视频编码模块（`agent/h264_encoder.py`）
- 可选 H.264 编码替代 JPEG
- 预计额外节省 70% 带宽
- 需安装 OpenCV（可选依赖）

#### 8. 音频转发模块（`agent/audio.py`）
- 使用 PyAudio 捕获系统音频
- 支持 Linux/macOS/Windows
- 可选插件，默认不启用

#### 9. Agent v2.0 重写（`agent/agent.py`）

- 连接自动重试（指数退避）
- 多控制器并发支持

- 完整的性能追踪
