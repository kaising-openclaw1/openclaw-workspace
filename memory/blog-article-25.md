## 一、Deepfake 图像的"破绽"在哪里？

### 1.1 频域破绽（最核心的检测手段）

真实相机拍摄的照片，其像素值经过离散余弦变换（DCT）后，频域系数呈**自然衰减分布**——低频成分强，高频成分弱，过渡平滑。

但 AI 生成图像（尤其是 GAN 和扩散模型）在频域中会表现出**异常峰值**：

```
真实照片频域：  ████████░░░░░░░░░░░░░  (平滑衰减)
AI 生成频域：   ████░░░████░░░░█░░░░░  (异常峰值)
```

这是因为生成模型在训练过程中对某些频率模式存在系统性偏差。这个偏差人类肉眼看不到，但数学上一目了然。

### 1.2 EXIF 元数据缺失

真实照片通常包含完整的 EXIF 信息：
- 相机型号、镜头参数
- 拍摄时间、GPS 坐标
- 光圈、快门速度、ISO

而 AI 生成图像：
- **要么完全没有 EXIF 数据**
- **要么包含矛盾的数据**（比如同时标注 iPhone 和 Canon 的标签）
- **要么暴露生成工具**（如 "Generated by DALL·E"）

### 1.3 噪声分布不自然

真实照片的噪声遵循**泊松-高斯混合分布**，这是相机传感器的物理特性决定的。

AI 生成图像的噪声：
- 在平滑区域过于均匀（"塑料感"）
- 在边缘处出现不自然的过渡
- 缺乏真实传感器的随机噪声模式

## 三、核心模块实现

### 3.1 频域分析模块

```python
import numpy as np
from scipy import fftpack
from PIL import Image

class FrequencyAnalyzer:
    """频域分析 - 检测 AI 生成图像的 DCT 异常"""
    
    def __init__(self, block_size=8):
        self.block_size = block_size
    
    def analyze(self, image_path: str) -> dict:
        """分析图像的频域特征"""
        img = Image.open(image_path).convert('L')  # 转灰度
        pixels = np.array(img, dtype=np.float64)
        
        # 分割为 8x8 块，计算每块的 DCT
        dct_coeffs = self._compute_block_dct(pixels)
        
        # 分析 DCT 系数分布
        anomaly_score = self._detect_anomalies(dct_coeffs)
        
        # 检测高频异常峰值
        peak_score = self._detect_frequency_peaks(dct_coeffs)
        
        return {
            'anomaly_score': float(anomaly_score),
            'peak_score': float(peak_score),
            'is_suspicious': (anomaly_score + peak_score) / 2 > 0.6
        }
    
    def _compute_block_dct(self, pixels: np.ndarray) -> np.ndarray:
        """分块计算 DCT 系数"""
        h, w = pixels.shape
        bs = self.block_size
        
        # 裁剪到块大小的整数倍
        h_crop = (h // bs) * bs
        w_crop = (w // bs) * bs
        pixels = pixels[:h_crop, :w_crop]
        
        # 重塑为 (块数_h, 块数_w, bs, bs)
        blocks = pixels.reshape(h_crop // bs, bs, w_crop // bs, bs)
        blocks = blocks.transpose(0, 2, 1, 3)
        blocks = blocks.reshape(-1, bs, bs)
        
        # 对每个块计算 2D DCT
        dct_blocks = np.array([fftpack.dct(fftpack.dct(block.T, norm='ortho').T, norm='ortho') 
                                for block in blocks])
        return dct_blocks
    
    def _detect_anomalies(self, dct_coeffs: np.ndarray) -> float:
        """检测 DCT 系数分布异常"""
        # 真实图像的 DCT 能量集中在低频
        # AI 生成图像在特定高频位置有异常能量
        dc = dct_coeffs[:, 0, 0]  # DC 系数
        ac = dct_coeffs[:, 1:, 1:]  # AC 系数
        
        # 计算能量比
        total_energy = np.sum(dct_coeffs ** 2, axis=(1, 2))
        high_freq_energy = np.sum(ac[:, 3:, 3:] ** 2, axis=(1, 2))
        ratio = high_freq_energy / (total_energy + 1e-10)
        
        # 真实照片的高频能量占比通常 < 5%
        mean_ratio = np.mean(ratio)
        return min(1.0, mean_ratio / 0.1)  # 归一化到 0-1
    
    def _detect_frequency_peaks(self, dct_coeffs: np.ndarray) -> float:
        """检测频域异常峰值"""
        avg_dct = np.mean(np.abs(dct_coeffs), axis=0)
        
        # 计算局部标准差，寻找异常尖锐的峰值
        from scipy.ndimage import uniform_filter
        local_mean = uniform_filter(avg_dct, size=3)
        local_std = np.sqrt(uniform_filter(avg_dct ** 2, size=3) - local_mean ** 2)
        
        # 峰值定义为超过局部均值 + 3倍标准差的点
        peaks = avg_dct > (local_mean + 3 * local_std + 1e-10)
        peak_ratio = np.sum(peaks) / peaks.size
        
        return min(1.0, peak_ratio * 100)  # 放大信号
```

**关键思路：** 分块 DCT 是 JPEG 压缩的基础，也是检测 AI 生成图像最经典的方法。GAN 生成器在 8×8 块边界处会留下可检测的周期性伪影。

### 3.2 EXIF 元数据检查

```python
import piexif
from datetime import datetime

class ExifChecker:
    """EXIF 元数据完整性检查"""
    
    # 真实照片通常包含的 EXIF 标签
    ESSENTIAL_TAGS = {
        piexif.ExifIFD.DateTimeOriginal,
        piexif.ImageIFD.Make,       # 相机品牌
        piexif.ImageIFD.Model,      # 相机型号
        piexif.ExifIFD.ExposureTime, # 曝光时间
        piexif.ExifIFD.FNumber,      # 光圈值
        piexif.ExifIFD.ISOSpeedRatings,  # ISO
    }
    
    # AI 生成工具的标志性标签
    AI_GENERATOR_SIGNS = [
        b'DALL·E', b'Midjourney', b'Stable Diffusion',
        b'Adobe Firefly', b'DALL-E', b'generated',
        b'AI-generated', b'合成'
    ]
    
    def check(self, image_path: str) -> dict:
        """检查图像 EXIF 数据"""
        result = {
            'has_exif': False,
            'completeness': 0.0,
            'ai_signals': [],
            'suspicious': False
        }
        
        try:
            exif_dict = piexif.load(image_path)
            result['has_exif'] = True
        except (ValueError, KeyError):
            result['suspicious'] = True
            result['ai_signals'].append('EXIF 数据完全缺失')
            return result
        
        # 检查完整性
        present_tags = set()
        for ifd_name, ifd_dict in exif_dict.items():
            if isinstance(ifd_dict, dict):
                present_tags.update(ifd_dict.keys())
        
        found = present_tags & self.ESSENTIAL_TAGS
        result['completeness'] = len(found) / len(self.ESSENTIAL_TAGS)
        
        # 检查 AI 生成标记
        exif_bytes = str(exif_dict).encode()
        for sign in self.AI_GENERATOR_SIGNS:
            if sign in exif_bytes:
                result['ai_signals'].append(f'发现 AI 生成标记: {sign.decode()}')
                result['suspicious'] = True
        
        # 检查完整性异常
        if result['completeness'] < 0.3:
            result['ai_signals'].append('EXIF 数据严重不完整')
            result['suspicious'] = True
        
        # 检查时间戳合理性
        if piexif.ExifIFD.DateTimeOriginal in present_tags:
            try:
                date_str = exif_dict['Exif'][piexif.ExifIFD.DateTimeOriginal]
                if isinstance(date_str, bytes):
                    date_str = date_str.decode()
                dt = datetime.strptime(date_str, '%Y:%m:%d %H:%M:%S')
                if dt > datetime.now():
                    result['ai_signals'].append('EXIF 时间戳在未来')
                    result['suspicious'] = True
            except (ValueError, TypeError):
                result['ai_signals'].append('EXIF 时间戳格式异常')
        
        return result
```

### 3.3 噪声分布分析

```python
import numpy as np
from scipy import stats
from PIL import Image

class NoiseAnalyzer:
    """噪声分布一致性分析"""
    
    def analyze(self, image_path: str) -> dict:
        """分析图像的噪声分布特征"""
        img = Image.open(image_path).convert('L')
        pixels = np.array(img, dtype=np.float64)
        
        # 估计噪声（用原始图减去高斯模糊后的残差）
        from scipy.ndimage import gaussian_filter
        smoothed = gaussian_filter(pixels, sigma=2.0)
        noise = pixels - smoothed
        
        # 分析噪声分布
        noise_mean = np.mean(noise)
        noise_std = np.std(noise)
        
        # 正态性检验（真实照片噪声接近高斯分布）
        # 采样避免计算量过大
        sample = noise.ravel()[::4]
        if len(sample) > 5000:
            sample = sample[:5000]
        
        _, normality_p = stats.shapiro(sample[:min(5000, len(sample))])
        
        # 局部噪声一致性（AI 生成图在不同区域噪声不一致）
        local_stds = []
        block_size = 64
        for i in range(0, noise.shape[0] - block_size, block_size):
            for j in range(0, noise.shape[1] - block_size, block_size):
                block = noise[i:i+block_size, j:j+block_size]
                local_stds.append(np.std(block))
        
        consistency = np.std(local_stds) / (np.mean(local_stds) + 1e-10) if local_stds else 0
        
        return {
            'noise_mean': float(noise_mean),
            'noise_std': float(noise_std),
            'normality_p_value': float(normality_p),
            'local_consistency': float(consistency),
            'is_natural': normality_p > 0.01 and consistency < 0.5
        }
```

### 3.4 核心检测引擎（聚合所有信号）

```python
from dataclasses import dataclass
from typing import List, Optional

@dataclass
class DetectionResult:
    """单次检测结果"""
    file_path: str
    fake_probability: float  # 0.0 - 1.0
    anomaly_score: float     # 0.0 - 1.0
    signals: List[str]       # 检测到的异常信号
    details: dict            # 各模块详细结果
    verdict: str             # "likely_real" / "suspicious" / "likely_fake"

class DeepfakeDetector:
    """深度伪造检测器"""
    
    def __init__(self):
        self.freq_analyzer = FrequencyAnalyzer()
        self.exif_checker = ExifChecker()
        self.noise_analyzer = NoiseAnalyzer()
    
    def analyze(self, image_path: str) -> DetectionResult:
        """综合分析一张图像"""
        signals = []
        scores = []
        
        # 1. 频域分析
        freq_result = self.freq_analyzer.analyze(image_path)
        if freq_result['is_suspicious']:
            signals.append('频域异常：DCT 系数分布不符合自然图像特征')
            scores.append(0.4 * (freq_result['anomaly_score'] + freq_result['peak_score']))
        
        # 2. EXIF 检查
        exif_result = self.exif_checker.check(image_path)
        if exif_result['suspicious']:
            signals.extend(exif_result['ai_signals'])
            scores.append(0.3 * (1.0 - exif_result['completeness']))
        
        # 3. 噪声分析
        noise_result = self.noise_analyzer.analyze(image_path)
        if not noise_result['is_natural']:
            signals.append('噪声分布异常：不符合自然相机传感器特征')
            scores.append(0.3 * (1.0 - float(noise_result['normality_p_value'])))
        
        # 综合评分
        fake_prob = sum(scores) / max(len(scores), 1)
        
        if fake_prob > 0.7:
            verdict = 'likely_fake'
        elif fake_prob > 0.4:
            verdict = 'suspicious'
        else:
            verdict = 'likely_real'
        
        return DetectionResult(
            file_path=image_path,
            fake_probability=fake_prob,
            anomaly_score=np.mean(scores) if scores else 0.0,
            signals=signals,
            details={
                'frequency': freq_result,
                'exif': exif_result,
                'noise': noise_result
            },
            verdict=verdict
        )
```

## 四、REST API 服务

```python
from fastapi import FastAPI, UploadFile, File
from deepfake_guard import DeepfakeDetector
import tempfile
import os

app = FastAPI(title="DeepfakeGuard API")
detector = DeepfakeDetector()

@app.post("/detect")
async def detect(image: UploadFile = File(...)):
    """上传图像进行深度伪造检测"""
    with tempfile.NamedTemporaryFile(suffix=".jpg", delete=False) as tmp:
        content = await image.read()
        tmp.write(content)
        tmp_path = tmp.name
    
    try:
        result = detector.analyze(tmp_path)
        return {
            "file": image.filename,
            "fake_probability": round(result.fake_probability, 4),
            "verdict": result.verdict,
            "signals": result.signals,
        }
    finally:
        os.unlink(tmp_path)
```

```bash
python api_server.py --port 8080
```

## 七、局限性与改进方向

### 当前局限

1. **算法检测天花板**：纯算法方法对高质量伪造的检测率约 70-85%
2. **对抗样本脆弱**：攻击者可以通过后处理（压缩、裁剪、滤镜）降低检测率
3. **无法识别特定工具**：不能区分是用 Midjourney、DALL·E 还是 Stable Diffusion

### 改进方向

1. **引入深度学习分类器**：训练一个专门的真假图像分类模型（需要标注数据集）
2. **多模态检测**：结合文本-图像一致性检测（比如图片声称的地点和 EXIF GPS 对比）
3. **在线更新检测规则**：随着 AI 生成技术进步，持续更新检测算法
