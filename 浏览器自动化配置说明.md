## 使用方法

### CLI 命令

```bash
# 查看状态
openclaw browser status

# 启动/停止浏览器
openclaw browser start
openclaw browser stop

# 打开网页
openclaw browser open https://example.com

# 抓取页面快照（AI 可读格式）
openclaw browser snapshot

# 截图
openclaw browser screenshot

# 点击元素（使用快照中的 ref）
openclaw browser click e6

# 输入文本
openclaw browser type e3 "hello"

# 等待元素/文本
openclaw browser wait --text "Done"
openclaw browser wait "#main"
```

### Agent 工具调用

在对话中使用 `browser` 工具：

```json
{
  "action": "open",
  "url": "https://example.com"
}
```

```json
{
  "action": "snapshot",
  "refs": "aria"
}
```

```json
{
  "action": "act",
  "kind": "click",
  "ref": "e6"
}
```

## 进阶用法

### 1. 多配置文件

可以配置多个浏览器配置文件：

```json
{
  "browser": {
    "profiles": {
      "openclaw": { "cdpPort": 18800, "color": "#FF4500" },
      "work": { "cdpPort": 18801, "color": "#0066CC" },
      "remote": { "cdpUrl": "http://10.0.0.42:9222", "color": "#00AA00" }
    }
  }
}
```

使用时指定 `--browser-profile <name>`：
```bash
openclaw browser --browser-profile work open https://example.com
```

### 2. 远程 CDP

连接远程浏览器（如 Browserless、Browserbase）：

```json
{
  "browser": {
    "profiles": {
      "browserless": {
        "cdpUrl": "https://production-sfo.browserless.io?token=<API_KEY>",
        "color": "#00AA00"
      }
    }
  }
}
```

### 3. SSRF 安全策略

严格模式（仅允许公网访问）：
```json
{
  "browser": {
    "ssrfPolicy": {
      "dangerouslyAllowPrivateNetwork": false,
      "hostnameAllowlist": ["*.example.com", "example.com"]
    }
  }
}
```

## 故障排查

### 常见问题

**1. 浏览器无法启动**
```bash
# 检查配置
openclaw browser status

# 查看详细错误
openclaw browser start 2>&1
```

**2. 页面加载失败**
```bash
# 查看控制台错误
openclaw browser console --level error

# 查看网络请求
openclaw browser requests --filter api
```

**3. 元素找不到**
```bash
# 重新抓取快照（ref 会变化）
openclaw browser snapshot --interactive

# 高亮目标元素
openclaw browser highlight e12
```

### 调试模式

录制 Trace 进行深入调试：
```bash
openclaw browser trace start
# ... 执行操作 ...
openclaw browser trace stop
# 输出：TRACE:/tmp/openclaw/trace-xxx.zip
```

## 安全注意事项

1. **隔离性**: `openclaw` 配置文件是独立的，不会影响你的个人浏览器
2. **认证**: 浏览器控制服务绑定在本地回环地址（loopback）
3. **SSRF**: 默认允许内网访问，生产环境建议启用严格模式
4. **Token**: 远程 CDP URL 包含的 API 密钥不要提交到版本控制
